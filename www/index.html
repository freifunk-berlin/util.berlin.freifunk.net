Werkzeuge für Freifunk Berlin

Die Seite hält eine Sammlung nützlicher kleiner Werkzeuge für Freifunk Berlin bereit.

Knoteninfo

Dieses Tool erlaubt es unter Eingabe des Knotennamens zu den zugehörigen Zielseiten im Wiki, Monitor, OWM und Hopglass zu springen, falls diese existieren.

// https://util.berlin.freifunk.net/knoteninfo?knoten=zwingli-core&typ=wiki
// knoten (string): name of the node
// typ (enum, optional): wiki, monitor, owm, hopglass


Durchsuchbare Knotenliste

Dieses Tool stellt eine durchsuchbare Knotenliste mit Firmware-Versionen und Kontaktdaten bereit.

Kartenzentrierung

Dieses Tool nimm durch Komma getrennte Koordinaten entgegen und erlaubt es die OWM, OSM und GMap Karte auf diese zu zentrieren.

// https://util.berlin.freifunk.net/mapcenter?latlon=a,b&map=owm
// latlon (sting): latitude and longitude separated by ","
// map (enum): owm/osm/gmap


Firmware-Downloads

Der Freifunk Firmware Selector listet auf Basis der Eingabe eines Routermodels die passenden Firmware-Downloads nebst zugehöriger Prüfsummen auf.

Knoten von Karte entfernen

Dieses Tool löscht einen Node von der OWM und Hopglass-Karte indem es ein Updat mit leeren Informationen sendet. Es ist ausschließlich dazu gedacht zum Test aufgesetzte Knoten und dauerhaft entfernte Knoten zu löschen.

// https://util.berlin.freifunk.net/delete-node-from-map?name=example
// name (sting): name of the node

Node-Update

Berliner OWM-API-Endpoint (identisch zu api.openwifimap.net)

// http://util.berlin.freifunk.net/update_node/
